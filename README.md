## Installation
To run the portfolio on your machine: 
 ```bash
 git clone git@github.com:Hilary505/Portfolio.git 
 ```

Navigate to the project directory: 
```bash
cd Portfolio
 ```

Navigate to the project directory:
 ```bash
  npm start 
 ```

## Usage
Navigate to the project directory:
 ```bash
  node server.js
 ```
